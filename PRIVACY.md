
# Política de Privacidade — Guitar Architect

**Última atualização:** 24 de janeiro de 2026

O Guitar Architect respeita a sua privacidade. Esta Política de Privacidade descreve como as informações são tratadas quando você utiliza o aplicativo disponível em:

> [https://guitararchitect.com.br](https://guitararchitect.com.br)

---

## 1. Natureza do Aplicativo

O Guitar Architect é uma aplicação **frontend-only**, executada diretamente no navegador do usuário. Não há servidores próprios, banco de dados remoto ou sistemas de autenticação centralizada.

Isso significa que:

* O aplicativo não envia seus dados para servidores externos
* Não há contas online, senhas ou perfis armazenados remotamente
* Todo o funcionamento ocorre localmente no seu navegador

---

## 2. Dados que Coletamos

O Guitar Architect **não coleta dados pessoais**.

Mais especificamente, o aplicativo:

* ❌ Não solicita nome real
* ❌ Não solicita e-mail
* ❌ Não solicita telefone
* ❌ Não utiliza cookies de rastreamento
* ❌ Não utiliza ferramentas de analytics

---

## 3. Armazenamento de Dados (Local)

Os dados gerados pelo usuário — como projetos, diagramas, configurações e perfis locais — são armazenados exclusivamente por meio do **localStorage do navegador**.

### Como funciona:

* Os dados são salvos no próprio dispositivo do usuário
* Permanecem disponíveis mesmo após fechar o navegador
* São específicos para o navegador e computador utilizados

### Importante:

Os dados serão perdidos se o usuário:

* Limpar os dados do navegador
* Utilizar outro navegador ou outro computador
* Navegar em modo anônimo
* Reinstalar o sistema operacional

O Guitar Architect não possui meios técnicos para recuperar dados apagados localmente.

---

## 4. Sistema de Usuários (Perfil Local)

O recurso de "login" disponível no aplicativo funciona como um **perfil local**.

Isso significa que:

* O nome de usuário serve apenas para organizar e filtrar projetos no próprio navegador
* Não há autenticação remota
* Não há validação de identidade
* Não há compartilhamento de dados entre usuários

Cada perfil existe apenas no dispositivo onde foi criado.

---

## 5. Compartilhamento de Dados

O Guitar Architect **não compartilha dados com terceiros**, pois não coleta nem transmite informações para servidores externos.

---

## 6. Segurança

A aplicação é hospedada na plataforma Vercel, que fornece:

* Conexão segura via HTTPS (TLS)
* Proteção contra ataques de negação de serviço (DDoS)
* Infraestrutura de rede com firewall e CDN

Como não há armazenamento remoto de dados, os riscos de vazamento de informações pessoais são significativamente reduzidos.

---

## 7. Limitações de Responsabilidade

O Guitar Architect não se responsabiliza por:

* Perda de dados causada por limpeza do navegador
* Falhas no dispositivo do usuário
* Uso indevido do aplicativo por terceiros no mesmo computador

Recomenda-se que os usuários façam backups manuais de seus projetos quando o recurso de exportação estiver disponível.

---

## 8. Alterações nesta Política

Esta Política de Privacidade pode ser atualizada a qualquer momento para refletir melhorias no aplicativo ou mudanças legais.

A data da última atualização sempre será indicada no topo deste documento.
