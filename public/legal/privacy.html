Política de Privacidade

Última atualização: 24 de janeiro de 2026

O Guitar Architect respeita a sua privacidade. Esta Política de Privacidade descreve como as informações são tratadas quando você utiliza o aplicativo disponível em https://guitararchitect.com.br.

1. Natureza do Aplicativo

O Guitar Architect é uma aplicação frontend-only, executada diretamente no navegador do usuário. Não há servidores próprios, banco de dados remoto ou sistemas de autenticação centralizada.

  • O aplicativo não envia seus dados para servidores externos
  • Não há contas online, senhas ou perfis armazenados remotamente
  • Todo o funcionamento ocorre localmente no seu navegador

2. Dados que Coletamos

O Guitar Architect não coleta dados pessoais. Mais especificamente, o aplicativo:

❌ Não solicita nome real
❌ Não solicita e-mail
❌ Não solicita telefone
❌ Não utiliza cookies de rastreio

3. Armazenamento de Dados (Local)

Os dados gerados pelo usuário — como projetos, diagramas, configurações e perfis locais — são armazenados exclusivamente por meio do localStorage do navegador.

Importante:

Os dados serão perdidos se o usuário limpar os dados do navegador, utilizar modo anônimo ou trocar de computador. O Guitar Architect não possui meios técnicos para recuperar dados apagados localmente.

4. Sistema de Usuários (Perfil Local)

O recurso de "login" disponível no aplicativo funciona como um perfil local. O nome de usuário serve apenas para organizar e filtrar projetos no próprio navegador.

5. Compartilhamento de Dados

O Guitar Architect não compartilha dados com terceiros, pois não coleta nem transmite informações para servidores externos.

6. Segurança

A aplicação é hospedada na plataforma Vercel, que fornece conexão segura via HTTPS (TLS) e infraestrutura de rede robusta.

7. Contato

Para dúvidas, sugestões ou solicitações relacionadas à privacidade, entre em contato por:

E-mail: contato@guitararchitect.com.br
