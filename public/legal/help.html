Guitar Architect

Ajuda e FAQ

Guia Rápido e Dúvidas Frequentes

Última atualização: 15 de junho de 2026

English version in progress. This legal content is currently available in Portuguese only.

O Guitar Architect é uma workstation musical para visualizar, estudar, praticar e exportar ideias no braço de guitarras e baixos. O Guitar Architect reúne conta, sincronização em nuvem, migração de dados locais, conquistas, coleção de logos, instrumentos e backup por JSON.

Studio → Conta e Migração de Dados

1. Como funciona a conta Guitar Architect?

Você pode criar uma conta com e-mail e senha. A conta permite sincronizar seus dados entre dispositivos, incluindo projetos, coleção, conquistas, progresso, instrumentos cadastrados e preferências principais.

2. Ainda posso usar sem conta?

Sim. O app continua com funcionamento local. Porém, sem login, seus dados ficam vinculados ao navegador e podem ser perdidos se você limpar os dados locais. A conta sincronizada é o caminho recomendado.

3. O que é migrar dados locais?

É o procedimento para levar projetos antigos, criados com um nome livre como “EVH” ou “Visitante”, para sua nova conta sincronizada. O app detecta identidades locais neste navegador e mostra botões para mesclar esses dados na conta.

4. Como faço a migração?

Abra o app no mesmo navegador em que você usava seus projetos antigos, entre ou crie sua conta e clique em Migrar dados locais. Escolha o nome local encontrado e confirme a migração. O sistema mescla os dados e preserva o que for mais recente.

5. Preciso importar JSON para recuperar meus dados antigos?

Não necessariamente. Se os dados antigos ainda estiverem no mesmo navegador, use a migração local. O JSON continua sendo uma ponte segura quando os dados vieram de outro computador, outro navegador ou de outra pessoa.

6. O JSON deixou de existir?

Não. O JSON deixou de ser o principal meio de salvamento para usuários logados, mas continua importante como backup, portabilidade, arquivamento e troca manual de projetos.

Studio → Sincronização na Nuvem

7. O que é salvo na nuvem?

A conta sincronizada salva um snapshot com projetos do fretboard, instrumentos cadastrados, coleção de temas, conquistas, progresso, badge/logo ativo, idioma, tema visual e preferências principais.

8. Meus instrumentos também sincronizam?

Sim. “Meus Instrumentos” entra no snapshot da conta autenticada. Cadastros com fotos, especificações, valores e histórico de manutenção passam a acompanhar a conta, além do backup JSON próprio da área de instrumentos.

9. Como o app evita sobrescrever dados melhores?

Na sincronização, projetos e instrumentos são mesclados por data de atualização. O app tenta preservar o item mais recente quando encontra o mesmo identificador em fontes diferentes.

Studio → Diagramas e Navegação

10. Como começo um diagrama?

Se não houver diagramas na tela, clique em Criar Primeiro Diagrama. O botão acompanha a identidade ativa: azul para guitarra de 6 cordas, verde para baixos e roxo para guitarras 7/8 ou tema desbloqueado.

11. Onde ficam Aprender, Praticar, Acordes, CAGED e Modos?

Esses recursos agora são páginas próprias acessíveis pelo cabeçalho e pelo Menu. Elas explicam os conceitos e enviam ações para o fretboard, em vez de ficarem escondidas em abas antigas da toolbar.

Studio → Conquistas e Tiers

12. O que são conquistas e recompensas?

Conquistas registram progresso musical: prática, exploração, harmonia, ritmo, consistência, CAGED, tríades, modos, acordes e uso do app. Recompensas podem liberar logos, badges e identidades visuais da coleção.

13. Como funcionam os tiers?

Os tiers seguem a linguagem do Guitar Architect: T0 Candidato, T1 Aprendiz, T2 Pedreiro, T3 Contramestre, T4 Mestre de Obras, T5 Engenheiro e T6 Arquiteto/Guitar Hero.

Studio → Logos e Exportação

14. As logos bloqueadas estão protegidas?

Visualmente, cards bloqueados mostram apenas silhueta/placeholder. Para proteção real contra acesso direto por URL, as logos futuras devem sair da pasta pública e ir para um bucket privado, liberado por link assinado após o desbloqueio.

15. Posso usar minha própria logomarca nas exportações?

Sim. Você pode subir uma logo própria no modal de conta/identidade. Essa imagem pode aparecer em exportações. No futuro, o fluxo ideal é armazenar essa logo no backend em vez de embutir tudo localmente.

16. Como exporto PNG, PDF e JSON?

PNG e PDF são formatos finais para aula, impressão e compartilhamento visual. JSON é backup editável e ponte de importação/exportação. Usuários logados têm sync automático, mas ainda podem salvar JSON quando quiserem.

Studio → Compartilhamento de Projetos

17. Posso abrir projeto de outra pessoa?

Sim, por enquanto o caminho é importar o JSON que a outra pessoa enviou. Esse fluxo continua útil para troca manual de estudos, aulas e materiais externos.

18. Qual será a melhor solução futura para compartilhar projetos?

Com backend, o ideal é criar compartilhamento por link, com permissões. Um projeto poderia ser privado, compartilhado em modo leitura, duplicável para a conta do aluno ou colaborativo. Isso evita depender de arquivos JSON para toda troca.

19. E para compartilhar instrumentos?

Instrumentos têm dados mais pessoais, como valor pago, número de série e fotos. O melhor caminho é compartilhar instrumentos por cópia controlada ou ficha pública sanitizada, removendo campos sensíveis por padrão.

Studio → Avisos e Dados Locais

20. O app avisa quando desbloqueio uma conquista?

Sim. Há um toast global discreto de conquista. Ele não deve disparar repetidamente para a mesma conquista já desbloqueada.

21. Limpar o navegador ainda apaga meus dados?

Se você estiver usando apenas modo local, sim. Se estiver logado e sincronizado, seus dados principais podem ser recuperados pela conta. Mesmo assim, backups JSON continuam recomendados para arquivamento importante.

Kids → Custom Shop

22. Os instrumentos que eu criar ficam salvos?

Sim. Os instrumentos criados no Custom Shop são armazenados localmente no navegador utilizado.

23. Se eu trocar de computador ou celular, meus instrumentos aparecem automaticamente?

Não. Atualmente os projetos do Custom Shop ficam vinculados ao dispositivo e navegador onde foram criados.

24. Os instrumentos são vinculados à minha conta do Studio?

Ainda não. O salvamento em conta é um recurso planejado para futuras versões.

25. Posso apagar um instrumento criado?

Sim. Os instrumentos podem ser removidos a qualquer momento pelo próprio usuário.

Teens → XP e Progressão

26. O que é XP?

XP (Experience Points) representa sua evolução dentro da área Teens. Conforme você pratica e completa desafios, novos ranks e conteúdos são desbloqueados.

27. Como ganho XP?

O XP é obtido ao concluir atividades educacionais e desafios das ferramentas Teens.

28. Posso ganhar XP repetindo infinitamente o mesmo desafio?

Nem sempre. Algumas atividades concedem XP apenas na primeira conclusão de cada desafio específico. Outras atividades focadas em prática contínua continuam concedendo XP normalmente.

29. Quais atividades concedem XP apenas uma vez por desafio?

Atualmente:

  • Leitura de Blueprint
  • Construtor de Acordes
  • Desafios de Riffs
  • Caça às Escalas

30. Quais atividades permitem ganhar XP continuamente?

Atualmente:

  • Batidas Populares
  • Rhythm Lab

Teens → Ranks

31. Como funcionam os ranks?

Os ranks representam seu progresso dentro da área Teens.

Exemplo atual:

  • Rookie
  • Runner
  • Architect
  • Neon Player

32. O que os ranks desbloqueiam?

Novos conteúdos, regiões, shapes, desafios e recursos educacionais podem ser liberados conforme o avanço de rank.

Teens → Shapes e Regiões

33. Por que alguns Shapes ou Regiões aparecem bloqueados?

Alguns conteúdos são liberados progressivamente para incentivar a prática gradual e evitar sobrecarga de informação.

34. Como desbloqueio novos Shapes?

Acumulando XP nas atividades da área Teens.

Conta e Sincronização

35. Meu XP fica salvo onde?

Atualmente o XP é armazenado localmente no navegador utilizado.

36. Se eu limpar os dados do navegador, perco meu XP?

Sim. Como o armazenamento é local, a limpeza dos dados do navegador pode remover seu progresso.

37. O XP já é sincronizado com a conta do Studio?

Ainda não. Hoje a progressão do Teens funciona localmente. A sincronização com contas do Studio é uma possibilidade futura.

Sobre o estágio atual do projeto

38. O Guitar Architect já está finalizado?

Não. O Guitar Architect está em desenvolvimento contínuo. Correções, novas ferramentas, conteúdos, desafios e formas de progressão são adicionados regularmente.
